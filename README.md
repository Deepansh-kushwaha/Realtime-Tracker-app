# 🌎 Real-time Location Tracker

A real-time location tracking application that allows multiple users to share and view their locations simultaneously on an interactive map.

## 📱 App Preview
![Real-time Tracker Preview](/sample.png)

## ✨ Features

- 🗺️ Real-time location tracking
- 🔄 Automatic location updates
- 👥 Multi-user support
- 🎯 High accuracy geolocation
- 🗺️ Interactive OpenStreetMap integration
- ⚡ WebSocket-based real-time updates

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Real-time Communication**: Socket.IO
- **Map**: Leaflet.js with OpenStreetMap
- **Template Engine**: EJS

## 📋 Prerequisites

- Node.js installed on your machine
- A modern web browser with geolocation support
- Internet connection

## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Deepansh-kushwaha/Realtime-Tracker-app.git
cd real-time-tracker
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
npm start
```

4. Access the application:
   - Open `http://localhost:3000` in your browser
   - Allow location access when prompted
   - Share the public URL with others to track each other's locations

## 💡 Use Cases

- 🚗 Fleet tracking
- 👪 Family location sharing
- 🏃‍♂️ Friend tracking during outdoor activities
- 🚴‍♀️ Cycling group coordination
- 🏢 Field workforce management

## 🔒 Privacy Note

This application requires location permissions to function. Make sure to:
- Only share the application URL with trusted users
- Use appropriate security measures when deploying publicly
- Be aware that all connected users can see each other's locations
